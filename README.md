# Tic-Tac-Toe

This is a simple 2 player game of tic-tac-toe using Pygame's GUI

## Installations
- Clone the repository
- run the `main.py` file

## Tools Used
* [<b>Pygame</b>](https://www.pygame.org/docs/)
